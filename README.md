## 📦 Projeto 1 — Dashboard de Logística RápidLog

Dashboard completo de controle de entregas desenvolvido em Excel com visual estilo BI.

### 🛠️ Tecnologias e Fórmulas
- `PROCV` com tabela de preços por cidade
- `ÚNICO` para listas dinâmicas sem repetição
- `ÍNDICE + CORRESP + MÁXIMO` para busca de destaques
- `CONT.SE`, `SOMASE` e `MÉDIASE` para resumos
- Fórmulas matriciais com `Ctrl + Shift + Enter`
- Tabela Dinâmica
- 4 tipos de gráficos: Barras, Pizza, Pareto e Linha

### 📈 O que o dashboard mostra
- Cidade com maior receita
- Motorista destaque do mês
- Maior frete registrado
- % de entregas concluídas
- Frete total por motorista
- Distribuição de status de entregas
- Análise de Pareto por cidade
- Evolução temporal de entregas
